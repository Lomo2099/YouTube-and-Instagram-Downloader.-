# 🎥 YouTube & Instagram Downloader

A clean, modern, and ad-free desktop application to download videos and audio from YouTube and Instagram. No sketchy websites, no pop-ups—just the media you want.

✨ Features
* *Modern UI:* Built with CustomTkinter for a sleek, dark-themed interface.
* *Dual Formats:* Download high-quality MP4 videos or extract MP3 audio.
* *Progress Tracking:* Real-time status updates and progress percentage.
* *Multi-Platform:* Supports both YouTube and Instagram links.
* *Threaded Downloads:* The UI remains responsive while your media downloads in the background.

## 🛠️ Prerequisites
Before running the app, ensure you have the following installed:
1. *Python 3.12+**
2. *FFmpeg:* Required for merging video/audio and converting to MP3.
   - *Windows:* [Download via Gyan.dev](https://www.gyan.dev/ffmpeg/builds/)
